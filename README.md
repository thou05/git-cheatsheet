# Git Cheat Sheet

## Basic commands
- `git add`
- `git commit -m "message"`
- `git log --oneline`
- `git remote add <name> <url>`
- `git push -u origin main`
- `git fetch`
- `git pull`

## Collaboration
- `git branch`
- `git switch <branch-name>` 
- `git branch -d <branch-name>` 
- `git merge <branch-name>`
-  `git diff --staged`
